5 HTML Tags I've Learned

Abbriviation tag <abbr>
Used for acronyms or abbreviations - when you hover over the word on the live page it shows the expanded name/fullname of the acronym or abbreviation.
Heading tag (ex.<h1> </h1>)
Used to create headings for websites; <h1 > is the main heading and each heading after that (h2, h3, h4, h5 , h6) is styled differently for each level, decreasing in font size.
Line Break tag <br>
This a singular tag creates a line break for content. It is recommended not to use it repeatedly in your code unless absolutely needed.
Image tags <img>
For displaying images on the webpage. It needs information from two attributes: source (src), which provides the relative or absolute path(the link to the image), and alternative text(alt), which provides a decription of the image if it doesnt load on a webpage.
Table tag<table> </table >
This is the main component to creating tables in HTML, the container. Addtional tags needed to form the rows and imput all the data:<caption ></caption > - for the description/label of table; <thead ></thead > - for the header of the table; < tbody> </tbody > - for the body of the table; <tr ></tr > - for adding rows; and so on...
